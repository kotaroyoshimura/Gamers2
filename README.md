# Gamers

## サイト概要
〜新たなゲームとの出会いをこの場所から〜  
自分の遊んだゲームの感想を投稿することのできるサイト

### サイトテーマ
ゲームの感想を投稿することのできるレビューサイト

### テーマを選んだ理由
ブログ等でレビューしているサイト等は見かけるがそれをまとめて見ることのできるサイトがあれば便利だと考えたため

### ターゲットユーザ
若者中心

### 主な利用シーン
新しいゲームを始めたいと思った時
